## 📝 学习方法

### 遵循用户指导
> "先全面的认识所有工艺，再去钻研每个工艺的细节。在学习的时候要找一些产品流程实例结合去看，更容易理解单步工艺以及前后工序的联系。"

### 具体方法
1. **先概览后细节**: 每章先看概述，再深入细节
2. **结合实例**: 用 CMOS 工艺流程作为主线
3. **绘制图表**: 每个工艺步骤都绘制流程图
4. **建立联系**: 标注前后工序的依赖关系
5. **定期复盘**: 每天结束时总结当天学习内容
