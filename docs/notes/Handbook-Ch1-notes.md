## 1. 半导体产业发展历程

### 关键里程碑

| 年份 | 事件 | 意义 |
|------|------|------|
| 1947 | Bell Labs 发明点接触晶体管 | 半导体时代开端 |
| 1958 | Jack Kilby (TI) 发明 IC | 集成电路诞生 |
| 1960 | 平面工艺发明 | 可量产 IC 制造 |
| 1965 | Gordon Moore 提出摩尔定律 | 产业发展预测 |
| 1970 | Intel 4004 (10μm) | 第一款商用微处理器 |
| 1980 | 1μm 工艺节点 | VLSI 时代 |
| 1990 | 0.5μm 工艺 | ULSI 时代 |
| 2000 | 180nm → 130nm | Cu 互连引入 |
| 2007 | 45nm | High-k/Metal Gate |
| 2011 | 22nm | FinFET 3D 晶体管 |
| 2018 | 7nm | EUV 光刻引入 |
| 2022 | 3nm | GAA 晶体管 |

### 工艺节点演进规律
- **1970-2000**: 每 3 年缩小 0.7x
- **2000-2010**: 每 2 年缩小 0.7x
- **2010-至今**: 每 2-3 年缩小，速度放缓

## 2. IC 制造基本流程

### 前端工艺 (FEOL)
```
硅片制备 → 氧化 → 光刻 → 刻蚀 → 离子注入 → 退火 → 薄膜沉积 → CMP
```
**目标**: 在晶圆上制造晶体管

### 后端工艺 (BEOL)
```
接触孔形成 → 阻挡层/种子层 → 电镀 Cu → CMP → 多层互连 → 钝化层
```
**目标**: 互连晶体管形成电路

### 中道工艺 (Middle-End)
- 接触孔刻蚀
- 硅化物形成
- 应力工程

## 3. 晶圆尺寸演进

### 尺寸对比

| 参数 | 100mm (4") | 150mm (6") | 200mm (8") | 300mm (12") | 450mm (18") |
|------|-----------|-----------|-----------|-------------|-------------|
| 面积 (cm²) | 78.5 | 176.7 | 314.2 | 706.9 | 1590.4 |
| 引入时间 | 1970s | 1980s | 1990s | 2000s | 延期 |
| 相对产能 | 1x | 2.25x | 4x | 9x | 20x |
| 边缘损失 | 高 | 中 | 低 | 最低 | 极低 |

### 12" 晶圆优势
1. **单位芯片成本降低 30-40%**
2. **边缘芯片损失减少**: 周长/面积比更小
3. **自动化程度高**: 全程 FOUP 封闭传输
4. **产能规模效应**: 单厂月产 50K-100K 片

### 18" 晶圆挑战
- 设备投资过大 (~$15B/厂)
- 晶圆重量 (~70kg) 搬运困难
- 市场需求不足
- **现状**: 无限期延期，行业聚焦 300mm 优化

## 4. 摩尔定律与工艺缩放

### 摩尔定律定义
> "集成电路上可容纳的晶体管数量，约每隔 18-24 个月便会增加一倍，性能也将提升一倍。"
> — Gordon Moore, 1965

### 缩放比例 (Dennard Scaling)
| 参数 | 缩放比例 | 效果 |
|------|----------|------|
| 线宽 (W) | 0.7x | 密度 2x |
| 电压 (V) | 0.7x | 功耗不变 |
| 电流 (I) | 0.7x | 速度 1.4x |
| 电容 (C) | 0.7x | RC 延迟 0.7x |

### 缩放挑战 (28nm 以下)

**1. 短沟道效应 (SCE)**
- 沟道长度过短，栅极失去控制
- 亚阈值漏电增加
- **解决**: FinFET、GAA 等 3D 结构

**2. 栅极氧化层隧穿**
- SiO₂厚度 < 2nm 时漏电剧增
- **解决**: High-k 介质 (HfO₂) + Metal Gate

**3. 互连 RC 延迟**
- Al 电阻率高 → Cu 替代
- 低 k 介质减少电容

**4. 光刻分辨率限制**
- 193nm 浸没式光刻极限 ~38nm
- **解决**: EUV (13.5nm)、多重曝光

## 5. 关键技术趋势

### 5.1 FinFET (3D 晶体管)
- **引入节点**: 22nm (Intel, 2011)
- **结构**: 沟道竖立如鱼鳍，栅极三面包裹
- **优势**: 更好的栅极控制，漏电降低 90%
- **局限**: 3nm 以下静电控制不足

### 5.2 GAA (Gate-All-Around)
- **引入节点**: 3nm (Samsung, 2022)
- **结构**: 纳米片 (Nanosheet)，栅极四面包裹
- **优势**: 最佳静电控制，可独立调节宽度
- **变种**: MBCFET (Samsung), RibbonFET (Intel)

### 5.3 EUV 光刻
- **波长**: 13.5nm (vs 193nm 浸没式)
- **引入节点**: 7nm (TSMC, 2018)
- **优势**: 单次曝光 ~13nm 线宽，减少多重曝光
- **挑战**: 光源功率、光罩缺陷、成本 ($150M/台)

### 5.4 Chiplet 异构集成
- **概念**: 将大芯片拆分为小 Chiplet，先进封装集成
- **优势**: 良率提升、成本降低、混合工艺节点
- **技术**: 2.5D (CoWoS)、3D (Foveros)、UCIe 标准

## 6. 与 YMS 的关联

### 关键数据模型映射

| 工艺步骤 | YMS 数据模型 | 关键参数 |
|----------|-------------|----------|
| 光刻 | Measurement | CD 均匀性、套刻精度 |
| 刻蚀 | Measurement | 刻蚀速率、选择比、均匀性 |
| 薄膜 | Measurement | 厚度、折射率、应力 |
| 注入 | Measurement | 剂量、能量、薄层电阻 |
| CMP | Measurement | 去除速率、平坦度 |
| 全工艺 | Defect | 缺陷密度、缺陷类型分布 |

### SPC 监控重点
- **CD-SEM**: 关键尺寸控制 (±3σ < 5%)
- **薄膜厚度**: 均匀性 (±3σ < 1.5%)
- **缺陷密度**: 每层 < 0.1 defects/cm²
- **良率**: 晶圆级良率追踪 (Wafer Yield)

## 7. 核心公式

### 1. 摩尔定律数学表达
$$ N(t) = N_0 \times 2^{(t-t_0)/T} $$
- $N(t)$: t 时刻晶体管数量
- $T$: 翻倍周期 (18-24 个月)

### 2. 每片晶圆芯片数 (Gross Die Per Wafer)
$$ GDPW = \frac{\pi \times (D/2)^2}{S} - \frac{\pi \times D}{\sqrt{2 \times S}} $$
- $D$: 晶圆直径
- $S$: 芯片面积

### 3. 晶圆良率模型 (Poisson)
$$ Y = e^{-D \times S} $$
- $D$: 缺陷密度 (defects/cm²)
- $S$: 芯片面积 (cm²)
