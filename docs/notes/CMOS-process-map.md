## 2. 8 大工艺详细分解

### 2.1 硅片制备 (Wafer Preparation)

**输入**: 高纯度多晶硅 (99.9999999%, 9N)  
**输出**: 抛光晶圆 (Prime Wafer)

```
多晶硅 → 单晶生长 (CZ/FZ) → 切片 → 倒角 → 研磨 → 腐蚀 → 抛光 → 清洗 → 检验
```

**关键参数**:
| 参数 | 规格 | 测量方法 |
|------|------|----------|
| 电阻率 | 0.001-100 Ω·cm | 四探针 |
| 氧含量 | 5-20 ppma | FTIR |
| 碳含量 | <0.5 ppma | FTIR |
| 缺陷密度 | <0.1 defects/cm² | 表面扫描 |
| 平整度 (TTV) | <1μm | 激光干涉 |

**YMS 关联**: Wafer Info 表记录每片晶圆的原始参数

---

### 2.2 氧化 (Oxidation)

**输入**: 裸硅片  
**输出**: 氧化层 (SiO₂)

**反应方程式**:
```
干氧氧化：Si + O₂ → SiO₂  (质量好，速率慢)
湿氧氧化：Si + 2H₂O → SiO₂ + 2H₂  (速率快，质量一般)
```

**应用场景**:
| 氧化层类型 | 厚度 | 用途 |
|------------|------|------|
| 场氧 (FOX) | 400-800nm | 器件隔离 |
| 栅氧 (Gate Oxide) | 1-5nm | 晶体管栅极介质 |
| 掩蔽氧化 | 50-100nm | 注入掩蔽 |
| 牺牲氧化 | 10-20nm | 表面修复 |

**关键参数**: 厚度均匀性 ±1%、缺陷密度 <0.1 defects/cm²

---

### 2.3 光刻 (Photolithography)

**输入**: 涂胶晶圆 + 掩膜版  
**输出**: 光刻胶图形

**9 步流程**:
```
表面准备 → 涂胶 → 软烘 → 对准 → 曝光 → 后烘 (PEB) → 显影 → 硬烘 → 检验
```

**关键指标**:
| 参数 | 定义 | 规格 |
|------|------|------|
| CD (关键尺寸) | 最小图形宽度 | ±3σ < 5% |
| 套刻精度 | 层间对准误差 | <10nm (7nm 节点) |
| 焦深 (DOF) | 可接受聚焦范围 | ~100nm |
| 分辨率 | 最小可分辨图形 | R = k₁λ/NA |

**YMS 关联**: CD-SEM 数据、套刻测量数据

---

### 2.4 刻蚀 (Etching)

**输入**: 光刻胶图形晶圆  
**输出**: 材料图形

**分类对比**:
| 类型 | 原理 | 各向异性 | 选择比 | 应用 |
|------|------|----------|--------|------|
| 湿法刻蚀 | 化学溶液 | 各向同性 | 高 | 清洗、去胶 |
| 干法刻蚀 (RIE) | 等离子体 | 各向异性 | 中 | 图形刻蚀 |
| 干法刻蚀 (ICP) | 高密度等离子体 | 高各向异性 | 高 | 深硅刻蚀 |

**关键参数**:
- 刻蚀速率：100-1000 nm/min
- 选择比：>10:1 (对光刻胶/下层材料)
- 均匀性：±3%
- CD 偏差：<5nm

---

### 2.5 离子注入 (Ion Implantation)

**输入**: 硅片 + 掺杂剂离子  
**输出**: 掺杂硅片

**掺杂剂类型**:
| 类型 | 离子 | 能量范围 | 应用 |
|------|------|----------|------|
| P 型 | B⁺, BF₂⁺ | 1-50 keV | PMOS 源漏 |
| N 型 | P⁺, As⁺ | 10-500 keV | NMOS 源漏 |

**关键参数**:
- 剂量：10¹¹ - 10¹⁶ ions/cm²
- 能量：1 keV - 3 MeV
- 剂量均匀性：±1%
- 注入角度：0°- 60° (避免沟道效应)

**退火激活**:
- RTA (快速热退火): 1000-1100°C, 1-10 秒
- 激光退火：局部加热，减少扩散

---

### 2.6 薄膜沉积 (Thin Film Deposition)

**输入**: 晶圆 + 前驱体/靶材  
**输出**: 功能薄膜

**三大技术对比**:
| 技术 | 原理 | 速率 | 台阶覆盖 | 均匀性 | 典型应用 |
|------|------|------|----------|--------|----------|
| PVD | 物理溅射 | 快 | 一般 | ±3% | 金属层 |
| CVD | 化学反应 | 中 | 好 | ±1.5% | 介质层 |
| ALD | 自限制反应 | 慢 | 极好 | ±0.5% | High-k |

**薄膜类型**:
| 薄膜 | 材料 | 厚度 | 用途 |
|------|------|------|------|
| 栅介质 | SiO₂, HfO₂ | 1-5nm | 晶体管栅极 |
| 栅电极 | Poly-Si, Metal | 50-200nm | 晶体管栅极 |
| 侧墙 | Si₃N₄, SiO₂ | 10-50nm | 自对准工艺 |
| ILD | SiO₂, FSG | 200-1000nm | 层间介质 |

---

### 2.7 CMP (化学机械抛光)

**输入**: 非平坦晶圆  
**输出**: 全局平坦化晶圆

**抛光机理**:
```
化学作用：氧化剂氧化表面材料 → 形成可溶性化合物
机械作用：磨料颗粒研磨去除氧化层
协同效应：化学 + 机械 = 高效去除
```

**应用工艺**:
| 工艺层 | 抛光材料 | 去除速率 | 终点检测 |
|--------|----------|----------|----------|
| STI CMP | SiO₂ | 200-400 nm/min | 电机扭矩 |
| Poly CMP | Poly-Si | 150-300 nm/min | 电机电流 |
| W CMP | Tungsten | 100-200 nm/min | 光学终点 |
| Cu CMP | Copper | 200-400 nm/min | 电机电流 |

**关键缺陷**: Dishing、Erosion、Corrosion、Scratch

---

### 2.8 金属化 (Metallization)

**输入**: 带沟槽/通孔的晶圆  
**输出**: 金属互连结构

**大马士革流程**:
```
介质层沉积 → 光刻 → 刻蚀 → 阻挡层 (Ta/TaN) 
→ 种子层 (Cu) → 电镀 Cu → 退火 → CMP → 清洗
```

**多层互连**:
```
M8 (顶层，厚金属，全局时钟)
 ↓ Via
M7 (厚金属，电源)
 ↓ Via
M6 (中等，信号)
 ↓ Via
M5 (中等，信号)
 ↓ Via
M4 (薄金属，局部)
 ↓ Via
M3 (薄金属，局部)
 ↓ Via
M2 (薄金属，局部)
 ↓ Via
M1 (薄金属，接触晶体管)
 ↓ Contact
晶体管 (FEOL)
```

**关键参数**:
- Cu 电阻率：<2.0 μΩ·cm
- 电迁移寿命：>10 年 @105°C
- 粘附力：>5 J/m²

## 3. 工艺整合 - 典型 CMOS 流程

### 3.1 NMOS 晶体管制造 (简化版 23 步)

```
1.  硅片准备 (Starting Wafer)
2.  阱区注入 (Well Implant)
3.  隔离氧化 (STI Formation)
4.  栅氧生长 (Gate Oxide)
5.  多晶硅沉积 (Poly Deposition)
6.  光刻 - 栅极定义 (Gate Photo)
7.  栅极刻蚀 (Gate Etch)
8.  源漏注入 - N 型 (S/D Implant N)
9.  退火激活 (Anneal)
10. 侧墙形成 (Spacer Formation)
11. 硅化物形成 (Silicide)
12. 接触孔光刻 (Contact Photo)
13. 接触孔刻蚀 (Contact Etch)
14. 阻挡层沉积 (Barrier Deposition)
15. 钨沉积 (W Deposition)
16. 钨 CMP (W CMP)
17. M1 介质层沉积 (ILD Deposition)
18. M1 通孔光刻 (Via1 Photo)
19. M1 通孔刻蚀 (Via1 Etch)
20. 阻挡层/种子层 (Barrier/Seed)
21. 铜电镀 (Cu ECD)
22. 铜 CMP (Cu CMP)
23. 最终测试 (Final Test)
```

## 4. 关键工艺控制点 (KPP) 汇总

| 工艺 | 关键参数 | 控制规格 | 测量设备 |
|------|----------|----------|----------|
| 氧化 | 氧化层厚度 | ±1% | 椭偏仪 |
| 光刻 | CD 均匀性 | ±3σ < 5% | CD-SEM |
| 光刻 | 套刻精度 | <10nm | 套刻量测 |
| 刻蚀 | 刻蚀速率 | ±3% | 膜厚仪 |
| 刻蚀 | 选择比 | >10:1 | SEM |
| 注入 | 剂量均匀性 | ±1% | 四探针 |
| 薄膜 | 厚度均匀性 | ±1.5% | 椭偏仪 |
| CMP | 去除速率 | ±5% | 膜厚仪 |
| CMP | Dishing | <50nm | 轮廓仪 |
| 金属化 | Cu 电阻率 | <2.0 μΩ·cm | 四探针 |

## 5. YMS 数据模型映射

### 5.1 工艺 - 数据表映射

| 工艺步骤 | YMS 数据表 | 关键参数字段 |
|----------|-----------|-------------|
| 氧化 | measurement_oxide | thickness, uniformity, defect_count |
| 光刻 | measurement_photo | cd_mean, cd_uniformity, overlay_x, overlay_y |
| 刻蚀 | measurement_etch | etch_rate, selectivity, profile_angle |
| 注入 | measurement_implant | dose, energy, rs_uniformity |
| 薄膜 | measurement_film | thickness, refractive_index, stress |
| CMP | measurement_cmp | removal_rate, dishing, erosion |
| 金属化 | measurement_metal | resistance, em_lifetime, adhesion |
| 全工艺 | defect_detail | x, y, category, size, kill_ratio |

### 5.2 良率分析流程

```
数据采集 → 缺陷分类 → Pareto 分析 → SPC 监控 → 异常告警 
→ 根因分析 → 改进措施 → 效果验证 → 知识沉淀
```

## 6. 术语索引 (按工艺分类)

### FEOL 术语
- 氧化：Dry/Wet Oxidation, LOCOS, STI
- 光刻：Photoresist, Mask, Exposure, Development
- 刻蚀：RIE, ICP, Selectivity, Anisotropy
- 注入：Dopant, Dose, Energy, Anneal, RTA

### BEOL 术语
- 薄膜：PVD, CVD, ALD, Step Coverage
- 金属化：Damascene, Barrier, Seed Layer, ECD
- CMP：Slurry, Pad, Dishing, Erosion

### 通用术语
- 良率：Yield, Defect Density, Kill Ratio
- 统计：SPC, Cp/Cpk, Control Chart
- 质量：Excursion, RCA, CAPA
